Overview

Welcome to CSC209H, Software Tools and Systems Programming! This course focuses on the interface between user programs and the operating system and is the gateway to advanced systems courses like operating systems (CSC369), networking (CSC358), security (CSC347), and databases (CSC343).

This term, we will study the memory model, system calls, and systems structures including files, sockets, and processes. We'll use the C language throughout the course, so by the end of the term, you should be comfortable developing programs in C that interact with the operating system.

The course policies, including the evaluation scheme and important dates/deadline can be found on the (tentative) course information sheet.

Course Structure

CSC209 is delivered as an "inverted" class. Each week, we will post videos, links to readings, and exercises on an online system (PCRS) that should be completed before class. These weekly "preparation" exercises are due Sunday night and are worth 5% of your mark.

During class, we will be doing hands-on activities to practice and extend what you learned from the online lectures. This will include lots of collaborative work with your classmates and time for questions and interaction with the instructor. In-class participation will be worth 3% of your mark.

Starting in tutorial -- and wrapping up by the end of the week -- you will complete small activities that build on the week's concepts. Some of these exercises will be hosted online, on PCRS, and others will need to be completed on a linux machine (like the ones in the computing labs). The TAs will be available to help get you started during tutorial. These weekly exercises are due on Friday and are worth 10% of your mark.

Separately, over the course of the term, you will complete four larger programming assignments. These programs will require that you use concepts from class, but will also ask you to practice designing programs and to use more sophisticated tools. These assignments, together, will be worth 32%.
